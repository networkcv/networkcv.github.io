First Blog

Category: Life

前言

这是我的第一个博客，从开始搭建到完成用了近两天的时间，其中遇到了很多问题，解决之后回过头来看还是挺简单的。

搭建这个博客的初衷是 记录学习 技术沉淀 记录自己的生活

使用 hexo + Chic + github page 来完成。

很喜欢chic这种简约的设计。

回顾博客的诞生

回想起来这个博客诞生的莫名其妙又顺理成章，起由是工作上最近要使用Git，之前一直用svn，于是打算学习一波，在学习Git常用命令中需要记笔记，以往都是用txt文件，通过换行来区分这种方式虽然高效但是可读性不是很好，这次突然想换个其他的方式来试试，首先就想到了MarkDown，立马去学习MarkDown的基本语法。

完成git学习和笔记后，觉得markdown真的很赞，后悔这么晚才接触，没试过的朋友可以试试，然后打算练习git命令，就在github上建了个仓库push上去，顺便也将其他的学习笔记统统提交上去，做成了一个技术笔记的仓库。但是浏览起来不够直观，于是萌生了建一个博客的想法，刚好可以使用Github Pages 提供的免费服务，于是就有了这个博。

除了记录学习和技术沉淀外，再加上分享自己的生活。

博客之外的故事

在大学里就接触过Github和个人博客，当时忙着玩游戏，对学习不那么上心，所以就随便注册了个账号，有一次朋友过生日，想送个别致的礼物。于是乎就打算用程序员的方式祝福——建几个静态页面、写点JS、再搞个好看的背景、配上音乐。

大一的时候有学过一些HTML和JS基础，不过写出来巨丑，自己都看不下去，于是就去网上扒了一些例子拿下来改最后的成品差不多是首页类似于黑客帝国的那种绿色代码雨，几秒后跳到一个全屏酷炫的登录系统页面,让她输入生日和名字后校验，通过后展示一个类似于下图的页面，附带烟花特效，加载时播放音乐。
